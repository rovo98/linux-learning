
# 关机和重启命令


#### shutdown 命令

- shutdown [选项] 时间
- 选项 ：
    -c  	取消前一个关机命令  cancle
    -h		关机
    -r		重启

注意 ：实际工作中，服务器一般是不使用远程关机的，最多远程重启

----------------------------------------------------------------------------------

#### 其他关机命令 

- halt
- poweroff
- init 0

存在问题 ： 实际开发中基本不使用

- reboot 重启

-----------------------------------------------------------------------------------

#### 概念引入 ： 系统运行级别
- 0 		关机
- 1		单用户
- 2		不完全多用户，不含NFS服务
- 3 		完全多用户
- 4		未分配
- 5		图形界面
- 6 		重启

-------------------------------------------------------------------------------------

- cat /etc/inittab
   修改系统默认运行级别

- runlevel
   查询系统当前运行级别

- logout  退出登录，
[注意]  每次操作结束后，需要通过此方式退出登录
